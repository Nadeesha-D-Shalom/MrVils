# MrVilz — Follow The Wild

A visually rich, animated landing page designed to promote the **MrVilz brand** across multiple social platforms.
This project focuses on **premium UI/UX**, smooth animations, and a strong identity inspired by a *Wild West aesthetic*.

---

## Overview

MrVilz is a modern single-page web experience that:

* Showcases a strong branded identity
* Provides direct links to social media platforms
* Encourages engagement through a creative concept
  **"One follow = One tree planted"**
* Delivers a cinematic and immersive visual experience

---

## Features

### UI / UX

* Clean, elegant, and premium layout
* Fully responsive design (mobile + desktop)
* Smooth animations and transitions
* Custom typography using Google Fonts

### Visual Effects

* Animated logo with halo and ring effects
* Background canvas rendering
* Scroll progress indicator
* Hover interactions on social cards
* Ripple click effects

### Branding Elements

* Custom logo display
* Animated brand name (MrVilz)
* Tagline and ornamental dividers
* Themed color palette (red, gold, parchment, green)

### Social Integration

Direct links to:

* Facebook
* Instagram
* TikTok
* YouTube

Each platform is presented with:

* Custom styled card
* Hover animations
* Platform-specific branding colors
